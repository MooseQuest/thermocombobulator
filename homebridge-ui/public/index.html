Thermostats

Create a thermostat for an area, then add the devices it controls (heat, A/C, fans). Each becomes one thermostat you can place in any HomeKit room. Sign-ins stay on your Homebridge — nowhere else.

Group thermostats

Optional. Make one thermostat that steers several rooms at once (e.g. Whole Home). It commands the room thermostats — it never fights them.

Save writes your setup to config. Then restart Homebridge (the banner that pops up, or the Homebridge menu) to apply it. The config editor's own Save button at the very bottom now does the same thing.
